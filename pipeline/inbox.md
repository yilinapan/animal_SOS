# 手動收件匣

滑 Threads 看到求助文 → 貼文右上角分享 → 複製連結 → 貼到下面。
連結的下一行可以貼內文（有內文 AI 才能自動分類；只有連結也行，會標成待人工判讀）。
一行連結算一則，commit 之後 workflow 會自動處理並清空這個檔案。
