# 子報告：台灣野生動物救傷痛點

研究日期：2026-06-11。重要說明：本次大量網頁的直接抓取（WebFetch/curl）遭伺服器以 403 阻擋，因此多數內容係透過搜尋引擎回傳的摘要取得。標籤：**[事實]**=有來源佐證的記載；**[軼事]**=個別陳述/論壇；**[推論]**=研究者判斷。正式引用前請開原始連結二次確認。

## 1. 野生動物急救站（農業部生物多樣性研究所／前特生中心，WRRC）

- **[事實]** 急救站於1993年12月22日成立，為國內首處野生動物救傷專責單位，迄今逾30年。[來源: https://www.tesri.gov.tw/WRRC ；https://e-info.org.tw/node/238744]
- **[事實]** 人力僅約19人，需同時承擔救傷、復健、收容、研究、生命教育等業務。早年報導：「19個人一年救治超過500隻動物」，每年救治約500–600隻傷病野生動物，並長期照養逾1,000–2,000隻保育類動物。[來源: https://www.thenewslens.com/article/98002]
- **[事實]** 近年每年救治野生動物已成長至超過700–800隻。[來源: https://www.tbri.gov.tw/WRRC/ ；https://e-info.org.tw/node/238744]
- **[事實]** 經費吃緊：106年度林務局「野生物保育」計畫僅分配約6千多萬元，相較95年約2億8千萬元，少了2億多元。[來源: https://www.thenewslens.com/article/98002]（2017年前後報導，未核到原始預算書）
- **[事實]** 招聘困難：徵獸醫師條件含「適應遠離城市塵囂的鄉村生活」、假日輪班，起薪僅約新台幣3萬多元。[來源: 同上]
- **[軼事]** 獸醫師詹芳澤受訪表示業務量很重，照養2,000多隻動物、傷病送達時間不固定，「來了就開始忙起來」。[來源: 同上]
- **[事實]** 鳥類救傷約占急救站全部案例的三分之一；領角鴞是民眾最常撿拾的猛禽（多為學飛幼鳥）。[來源: https://www.moa.gov.tw/ws.php?id=2501743]
- **[事實]** 「零撲殺」政策後的犬貓攻擊統計：累積14,849筆原生種救傷案例中，594隻遭犬貓攻擊（犬425、貓169），約占4%；2017年零撲殺政策後攻擊案例顯著增加，經救傷後死亡及安樂死比例分別逾五成及七成。穿山甲救傷約2/3係遭遊蕩犬攻擊。[來源: https://e-info.org.tw/node/243369]
- **[推論]** 人為與外來威脅（路殺、犬貓、獸夾）造成的傷勢嚴重，救傷單位投入大量資源但實際野放成功率受限——這是「量能」之外更深層的痛點。

## 2. 野灣野生動物保育協會（WildOne，台東池上）

- **[事實]** 2016年發起、2017年立案，2020年8月15日成立台灣東部第一座非營利野生動物醫院，2024年屏東分部啟用。[來源: https://www.wildonetaiwan.org/ ；https://ubrand.udn.com/ubrand/story/12116/7614097]
- **[事實]** 痛點核心：東部長年缺乏專業救傷單位，動物受傷常須長途跋涉至他縣市，延誤黃金治療期。[來源: https://www.zeczec.com/projects/WildoneNGO]
- **[事實]** 年度救傷量（自陳）：2022年救傷658隻、野放157隻；2023年救傷512隻、野放163隻；2024年救傷524–532隻、92種物種、保育類153件，野放173隻、野放率約33%，約四成因傷重不治或預後不佳安樂死。最常見救援原因：失親落巢、車禍、不明原因異常。[來源: https://www.wildonetaiwan.org/news/702]（524 vs 532 可能為案件數 vs 個體數差異，未核原文）
- **[事實]** 成立至今累積救援逾2,200隻，含台灣黑熊、穿山甲、食蟹獴、長鬃山羊、山羌。2023年與林保署共同救援7隻台灣黑熊。[來源: https://www.wildonetaiwan.org/story/476]
- **[事實]** 經費困境：2023年約76.8%收入來自個人及企業捐款。[來源: https://neticrm.tw/news/8092]
- **[事實]** 2021年嘖嘖募資「有你相挺，重返山林」達標，3,020人支持。[來源: https://www.zeczec.com/projects/WildoneNGO]
- **[軼事/事實]** 人力吃緊：早期人員開自己的車救援；正職僅4人「非常吃緊」；野放訓練籠舍多為臨時帳篷。[來源: 同上]
- **[推論]** 野灣是「單一NGO撐起整個東台灣救傷量能」的脆弱結構：收入七成以上靠捐款、正職僅4人，募款下滑即直接衝擊營運。

## 3. 台北市野鳥學會救傷中心及各地鳥會（雛鳥季高峰）

- **[事實]** 北鳥救傷中心電話 (02)8732-8891，服務時間週一至週五12:00–20:00。[來源: https://www.wbst.org.tw/ambulance-birds]
- **[事實]** 收案量逐年攀升：1992年84隻 → 近年每年平均3,500隻以上；另有資料稱每年逾4,000隻。[來源: https://www.bird.org.tw/basicpage/117]
- **[事實]** 季節性暴量：3–9月繁殖季最忙；5–7月收到的傷鳥約占全年一半，主因大量學飛幼鳥被民眾撿拾。2013年所收傷鳥60%是民眾撿來的雛幼鳥。[來源: https://e-info.org.tw/node/211817 ；https://gd-park.org.tw/news/1212]
- **[事實/軼事]** 救傷中心僅約一名全職、一名兼職照養員＋志工；有志工在家照顧七、八隻鳥；繁殖季常「負荷不來」。[來源: https://e-info.org.tw/node/211817]
- **[事實]** 鳥會公開聲明「請體諒本社團無人力及經費外出接鳥」，呼籲民眾優先電話諮詢、勿隨手撿雛鳥。[來源: https://www.bird.org.tw/basicpage/117]
- **[推論]** 未找到「北鳥救傷中心整體暫停服務」的新聞；現況為「服務縮減/不外出接鳥+要求自行送達或先諮詢」。桃園鳥會另設「非營利野生動物診所」。[來源: https://taoyuanbird.org.tw/non-profit-wildlife-clinic/]

## 4. 民眾常見的錯誤行為

- **[事實]** 最常見錯誤：撿走正在學飛、不需救援的幼鳥。幼鳥離巢學飛屬正常現象，親鳥多在附近照顧；無受傷的幼鳥移到貓狗孩童不易接觸的矮枝、離開10–30分鐘觀察親鳥是否接走即可。「愛心變綁架」是救傷單位反覆強調的訊息。[來源: https://e-info.org.tw/node/37240 ；https://www.tcapo.gov.taipei/News_Content.aspx?n=67993BA7C146BC76]
- **[事實]** 雛鳥/幼鳥區分：雛鳥幾乎無毛、無法站立，最佳處置是儘快放回上方鳥巢；幼鳥已長羽、會跳會拍翅，通常不需介入。[來源: https://e-info.org.tw/node/37240]
- **[事實]** 餵食錯誤致死率高：不能餵麵包、牛奶、人類奶粉；幼鳥無法消化整顆小米；餵水可能造成失溫、傷口感染或嗆入氣管窒息。[來源: https://e-info.org.tw/node/37240 ；https://www.peopo.org/news/5075]
- **[事實]** 物種誤判：台灣夜鷹幼雛常被誤認「落巢需救援」而被撿走，造成親子離散；也有被誤認外來種而遭滅巢的案例。[來源: https://jalife.cafe/1035/ ；https://wuo-wuo.com/report/instantnews/1746-20230424]
- **[事實]** 領角鴞幼鳥落巢常被「好心撿拾飼養」；落巢幼鳥會自行跳回枝頭，民眾應就近防護貓狗/車輛而非帶走。[來源: https://zoo.kcg.gov.tw/zh-tw/news/125/]
- **[軼事]** PTT/Dcard 常見「先撿先養再求助」的經驗分享。[來源: https://www.ptt.cc/bbs/Aves/M.1395862224.A.5F4.html ；https://www.dcard.tw/f/talk/p/255933120]
- **[事實]** 遇到蛇：多數蛇無毒；室內先堵在原處再通報，可撥1999或119轉接，由消防或農業局委託專業廠商處理。[來源: https://www.hsinchu.gov.tw/News_Content.aspx?n=141&s=90124]

## 5. 民眾向政府通報野生動物時會發生什麼（權責問題）

- **[事實]** 通報管道：1999市民專線、1959全國動物保護專線（24小時）、林業及自然保育署專線 0800-000930。[來源: https://www.tcapo.gov.taipei/ ；https://www.moa.gov.tw/]
- **[事實]** 權責不清是結構性痛點：地方動保處主要處理犬貓；原生/保育類野生動物常由農業處森保科主責。各縣市分工不一——台北市動保處包辦所有動物捕捉，新竹縣動保所僅做犬貓，野生動物歸農業處森保科；非犬貓的珍奇寵物若非保育類也非原生種，易落入「三不管地帶」。[來源: https://wuo-wuo.com/topics/forshow/escape/1912-exotic-pet_03]
- **[事實]** 「捕蜂捉蛇」業務歸屬爭議：《消防法》未賦予消防員捕蜂捉蛇任務；2017年行政院協商決議由農政單位主政、消防適時支援；立即危害人命或夜間10時後緊急蜂蛇勤務由消防受理。[來源: https://e-info.org.tw/node/202379 ；https://www.agriharvest.tw/archives/18684]
- **[事實]** 消防員負擔失衡個案：「一年火災出勤90件、捕蜂捉蛇高達8000次」。[來源: https://www.thenewslens.com/article/81738]（個案數字，非全國統計）
- **[推論]** 對民眾最直接的痛點是「打了電話被踢皮球」：動保處說是野生動物找農業局/林業署，119/1999 互轉，農政與消防皆自陳人力技能不足——通報體驗破碎，延誤救援。

## 6. 重複性問題占用人力、季節暴量、野放/安樂死率

- **[事實]** 季節暴量：鳥類5–7月約占全年收案一半。[來源: https://e-info.org.tw/node/211817]
- **[事實]** 重複性諮詢消耗人力：鳥會明言「無人力及經費外出接鳥」並要求先電話諮詢；各單位反覆製作「撿到鳥怎麼辦」衛教，正因同樣問題每年重複大量湧入。（未找到量化統計）
- **[事實]** 野放/安樂死率：野灣2024野放率約33%、約40%傷重不治或安樂死；急救站犬貓攻擊案例死亡逾五成、安樂死逾七成；猛禽研究會2024傷鳥以領角鴞最多。[來源: https://raptor.org.tw/rescue/content/168]
- **[推論]** 核心痛點三層：(1) 民眾端知識落差→錯誤撿拾與餵食，製造大量可避免的案例與重複諮詢；(2) 通報端權責破碎→延誤與民怨；(3) 機構端量能與經費結構性不足。三者環環相扣，**源頭的衛教與通報整合最具槓桿效益**。
