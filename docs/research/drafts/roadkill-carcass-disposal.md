## 一、路殺社（台灣動物路死觀察網 TaiRON）

**成立與性質**
- 2011年8月在 Facebook 成立，由特有生物研究保育中心（現農業部生物多樣性研究所）助理研究員林德恩發起，為全世界首創利用社群媒體蒐集路死資料的公民科學社團。[文件事實，來源: https://zh.wikipedia.org/zh-tw/台灣動物路死觀察網 ；https://research.sinica.edu.tw/te-en-lin-roadkill-specimen/]
- 與中研院資訊所莊庭瑞副研究員合作建置官網「台灣動物路死觀察網」以儲存高解析度照片。[文件事實，來源: https://research.sinica.edu.tw/chuang-tyng-ruey-roadkill-creative-commons/]

**通報如何運作（FB社團、網站、APP）**
- 上傳流程：開啟手機 GPS 與相機「照片地理標識（地理標記）」功能 → 拍清晰照片 → 用樹枝將動物大體移到路旁草叢 → 回家後用線上回報系統上傳，系統自動讀取照片日期與座標，上傳後回傳一組流水號供存查。[文件事實，來源: https://roadkill.tw/page/233 ；https://roadkill.tw/participate/upload]
- APP 歷史摩擦：曾推出手機 App 並同步上傳 FB 與網站，但因 Facebook 頻繁變更 API 造成 App 常當機，遂改開發 Web App；舊版 iOS/Android App 於2017年永久停用並下架，改用線上回報系統（Web App 2.0）。[文件事實，來源: https://roadkill.tw/news/20170218 ；https://roadkill.tw/download/app]
- 痛點推論：通報需手機開啟地理標記、拍清晰照、事後再上傳，對一般民眾而言步驟偏多；APP 因第三方平台 API 變動而被迫下架，反映依賴 FB 生態的脆弱性。[推論，依據上列來源]

**標本寄送摩擦**
- 撿拾者將標本裝雙層夾鏈袋、註記採集資訊與線上回報流水號，外加紙箱或保麗龍箱，以 7-11「低溫宅配」+「貨到付款」寄至南投集集特生中心。[文件事實，來源: https://research.sinica.edu.tw/te-en-lin-roadkill-specimen/ ；FB 公告]
- 痛點推論：需冷凍、包裝、宅配等多道工序，門檻明顯高於單純拍照通報，故撿拾標本筆數遠低於通報筆數。[推論]

**參與規模與資料量（數字隨時間變動，各來源不一，並列如下）**
- 環境資訊中心（第七年報導）等指：會員約1.6萬–1.9萬名不等，上傳資料逾10萬筆，每年可蒐集上萬筆。[文件事實，來源: https://e-info.org.tw/node/211125 ；https://research.sinica.edu.tw/te-en-lin-roadkill-specimen/]
- TechNews（2019）：累積約15萬筆動物屍體大數據。[文件事實，來源: https://technews.tw/2019/12/08/taiwan-roadkill-data-becomes-precious-treasure/]
- 官網焦點計畫頁（WebSearch 摘要稱截至2026年數據）：超過10,156位成員、其中1,933位為資料貢獻者、累計上傳逾33,991筆、撿拾採集逾4,816件標本。[文件事實但待核，來源: https://roadkill.tw/about/about ；https://roadkill.tw/species] — 註：此數字與「逾10萬筆」報導落差大，疑為單一焦點物種或特定子集統計，務必開原頁確認口徑。

**資料用途**
- 狂犬病監測：2013年台灣睽違50年再現動物狂犬病，路殺社累積的鼬獾死亡資料與標本協助快速釐清疫情。2011年至2015年11月記錄477筆路死食肉目（鼬獾最多304筆），採集188件標本（鼬獾137件）送特生中心，約佔記錄39.4%；6.5年間公眾系統提供至少709件哺乳類標本；至2019年路殺社送驗標本約佔全台鼬獾檢驗量近50%。[文件事實，來源: https://research.sinica.edu.tw/te-en-lin-roadkill-specimen/]
- 狂犬病樣本撿拾與送驗工作後來「回歸各縣市防疫機關」（階段性任務完成）。[文件事實，來源: https://roadkill.tw/news/190136]
- 石虎、穿山甲、陸蟹、鳥類、兩棲爬蟲類減緩路殺為核心目標；穿山甲、石虎屍體屬野生動物產製品/保育類，撿拾後須送交主管機關。[文件事實，來源: https://roadkill.tw/news/197893]

## 二、路上動物屍體誰來收 — 民眾經驗

- 通報管道：撥打 **1999**，由地方政府環保局清潔隊到場收運。早年動物屍體被當一般廢棄物（環保局權責），後因可能是走失寵物，環保局改與動保處/動物防疫機關合作。[文件事實，來源: https://www.bobo.care/.../140239 ；https://www.ptepb.gov.tw/ ；臺南消防局 https://119.tainan.gov.tw/News_Content.aspx?n=25498&s=7691400]
- 分工：清潔隊現場收運 → 確認/掃描晶片 → 有晶片交動保處通知飼主除戶領回，無領回則火化樹葬；無晶片由動保處集體火化。[文件事實，來源: 新北市府；臺北市環保局 SOP]
- 民眾困惑/拖延（痛點）：論壇有「處理一隻死在路中間的貓也可以拖一個小時」、「路上動物死屍要如何處理」等抱怨，反映到場時效與權責管道對一般人不直觀。[軼事，來源: mobile01 t=3166768、t=652446；Dcard p/228981682]
- 動保團體觀點：有 FB 貼文呼籲「遇流浪動物屍體盡可能不要通報清潔隊或1999，否則屍體會當垃圾清運」，反映民間對「當垃圾處理」的不滿與對善終的期待。[軼事]
- 早年實況：屍體被當一般廢棄物處理曾引發爭議，促成高雄、臺北等地推「街頭曝屍犬貓不再當垃圾」、訂SOP。[文件事實，來源: https://udn.com/news/story/7470/5296366 ；https://e-info.org.tw/node/215588]
- 註：公路局/高公局之國道、快速道路權責本研究未取得直接佐證頁面。[資料不足，未證實]

## 三、寵物大體處理：清潔隊規則與民間殯葬

**公部門收費（各市不一）**
- 臺北市動保處委託焚化：5公斤以內 NT$350，超過每公斤加收70元；非設籍者2倍計。臺北市環保局街道犬貓 SOP「僅針對街道意外死亡犬貓」，家中寵物須民眾自費委託殯葬業或動保處。[文件事實，來源: https://www.tcapo.gov.taipei/cp.aspx?n=1F425FA2E64328D4]
- 桃園市動保處：約90元/公斤；並有「桃園市街道犬貓遺體處理作業流程」「犬貓屍體焚化委託服務」。[文件事實]
- 臺中市：有「動物屍體處理流程圖」區分免費/付費、一般廢棄物模式/仿人類模式；家中寵物可委託各區清潔隊或動物之家（須先預約）。[文件事實]
- 痛點推論：各縣市規則、費率、是否可交清潔隊、免費/付費門檻不一致，民眾難以一眼掌握。[推論]

**民間寵物殯葬費用與亂象**
- 費用區間：公部門最便宜（約200–1000元級距）；民間個別火化約 NT$4,000–7,000，集體火化約 NT$1,500–4,500（亦有報導4千多到8千多）。[文件事實]
- 亂象/糾紛：低價陷阱、骨灰混雜、儀式倉促、未立案業者多。報導稱新北市10家寵物生命紀念禮儀公司僅1家合法。[文件事實/軼事，來源: https://udn.com/news/story/7325/7179227]
- 主管機關回應：農業部動保司預計提出寵物殯葬定型化契約以建立管理制度。[文件事實，來源: https://udn.com/news/story/7470/8707180]

**法規漏洞（無法可管）**
- 現行《殯葬管理條例》以「人」為主體，寵物不在規範範圍；《動物保護法》僅規範生前對待，對死後處置無規範；寵物遺體因而得依《廢棄物清理法》當「廢棄物」焚燒。[文件事實，來源: https://www.ly.gov.tw/Pages/Detail.aspx?nodeid=6590&pid=215278]
- 業者多以工廠、休閒農場登記遊走法律邊緣；非都市土地申設受10公頃限制等門檻被批窒礙難行。[文件事實]
- 進展：經濟部商業司2023年1月增列「寵物生命紀念業」營業項目；農業部2023年7月訂審查作業要點；目前僅桃園、台中、苗栗等有自治條例施行，高雄預計2026施行、台南審查中、新北近期才有草案。[文件事實]

## 四、民俗「死貓掛樹頭、死狗放水流」與公衛

- 由來：舊農業社會習俗。相信死貓埋土會變貓妖危害地方，吊樹頭風乾七七四十九天魂魄散去；死狗放水流則與來世轉生信仰有關。[文件事實]
- 公衛/環境問題：有礙觀瞻、臭氣滋生蚊蠅、放水流恐堵塞水溝污染環境；現仍偶見「死貓吊樹頭」案例。[文件事實/軼事]
- 法律：棄置動物屍體依《廢棄物清理法》第27條禁止，違者依第50條可處1,200至6,000元罰鍰。[文件事實，來源: 台東縣府]
- 與狂犬病連結（推論）：現行防疫要求民眾「發現死亡食肉目動物請通報動保處採檢送驗」，與「私自處理屍體（掛樹/放水流）」直接衝突——民俗處置會使狂犬病監測檢體流失。[推論]

## 五、石虎／穿山甲路殺數字

- 石虎路殺逐年新高（路殺社統計）：2020年18件、2021年25件、2022年32件、2023年41件、2024年45件（歷年新高）。[文件事實，來源: https://udn.com/news/story/12940/8484359 ；https://e-info.org.tw/node/220323]
- 另一報導：2022年31件追平最高紀錄，南投14件首度超過苗栗。[文件事實] — 註：與上條2022「32件」略有出入，疑統計截點不同，以路殺社官方為準。
- 累計地區分布（2011年起）：苗栗最多152隻、南投66隻、台中27隻、彰化18隻；8年75隻（2019年報導）。[文件事實]
- 全台石虎族群「剩不到500隻」。[文件事實，來源: https://e-info.org.tw/node/217415]
- 苗栗路殺熱點：縣道140、128、苗29、苗52；省道台3、台13、台13甲、西濱快速道路。[文件事實]
- 穿山甲：被列為路殺社焦點物種，屍體屬野生動物產製品須送主管機關；但未取得可驗證的穿山甲路殺逐年件數。[資料不足]

## 六、狂犬病監測背景

- 台灣1961年後無動物狂犬病，2013年7月再現，疫情侷限鼬獾、白鼻心等野生動物。[文件事實，來源: 衛福部]
- 防檢局2013年起檢測食肉目野生動物、犬貓、蝙蝠共13,014件，904件確診（鼬獾892例、白鼻心9例等）。[文件事實] — 註：此為某一時點累計，數字隨年度更新。

## 核心痛點總結

1. **通報路徑分裂**：路殺研究（路殺社，拍照+GPS+標本寄送）與屍體清運（1999/環保局清潔隊+動保處）是兩套不相通系統，民眾常分不清該打哪、該做什麼。[推論]
2. **通報門檻高**：拍照需開地理標記、標本需冷凍宅配，撿拾標本量遠低於通報量。[推論+文件事實]
3. **權責與時效**：屍體被當垃圾、到場拖延、城鄉差異是反覆出現的民怨。[軼事]
4. **費率不一致**：各市清潔隊/動保處收費規則與免費門檻分歧。[推論+文件事實]
5. **寵物殯葬無中央專法**：依《廢清法》當廢棄物，多數業者未立案、收費亂、糾紛多，自治條例僅少數縣市有。[文件事實]
6. **民俗 vs 公衛衝突**：掛樹頭/放水流既違《廢清法》，也使狂犬病監測檢體流失。[推論+文件事實]
