# 子報告：流浪動物救援、中途之家荒與 TNR 生態痛點

研究日期：2026-06-12。方法說明：多數台灣網站 WebFetch 回傳 403，內容主要取自 WebSearch 摘要，引用前請開原始連結核對。標籤：**[事實]**=有來源 URL 佐證；**[軼事]**=個別陳述/論壇；**[推論]**=研究者判斷。

## 一、現況

- **[事實]** 2017 年 2 月台灣全面實施公立收容所「零撲殺」（告別十二夜）。[來源: https://www.coa.gov.tw/theme_data.php?theme=news&sub_theme=agri&id=6797]
- **[事實]** 2018 年全台收容所平均犬隻容留比約 7 成、僅 4 縣市超收；至 2023 年平均容留比已近 9 成、8 個收容所超額收容。台北市動物之家 2023/3 犬貓在養量逾 1,000 隻（合理容量約 450 隻），犬隻超收率達 166%。[來源: https://www.twreporter.org/a/6-years-after-no-kill-policy-adopted-public-shelters（報導者）；數字據搜尋摘要]
- **[事實]** 2020 年農委會推估全國遊蕩犬 155,869 隻，較 2018 年增加 9,096 隻。[來源: 同上報導者專題 https://www.twreporter.org/topics/6-years-after-no-kill-policy-adopted]
- **[事實]** 零撲殺後入所模式改為「精準捕捉」：只捕捉追車、咬人等問題犬或急難個體——但這些正是最難送養的犬隻，收容所從中途站變成「終養站」。[來源: https://www.twreporter.org/a/6-years-after-no-kill-policy-adopted-public-shelters ；https://e-info.org.tw/node/217007]
- **[事實]** 全國公立收容所收容處理統計（含認養率、所內死亡率）公開於政府開放資料與動保資訊網，可作為 bot 的資料來源。[來源: https://data.gov.tw/dataset/41236 ；https://animal.moa.gov.tw/Frontend/PublicShelter]

## 二、痛點清單

### 1. 收容所爆滿 → 民眾「送不進去」
- **[事實]** 零撲殺一週年後即有縣市階段性爆籠（桃園、新竹縣、嘉義市顯著）；媒體以「十二夜變調！全台收容所爆量」報導。[來源: https://e-info.org.tw/node/217007 ；https://www.thenewslens.com/article/89133]
- **[事實]** 彰化唯一流浪犬收容所早已爆滿。[來源: https://news.ltn.com.tw/news/local/paper/1231650]
- **[推論]** 對撿到動物的民眾而言，後果是：收容所以「精準捕捉」為由不收健康遊蕩犬貓，撿到的人被迫自行處理——這是社群求助文大量出現的結構性原因。

### 2. 中途之家荒：「找不到中途」常態化
- **[事實]** 衛教文普遍承認：志願貓中途「在貓爆發的季節常常客滿，不一定能馬上接手」；找不到中途時的選項只剩自行收編、找朋友、上社群 PO 文、或付費請動物醫院暫時收留。[來源: https://www.catzoo.tw/blog/14520 ；https://www.ladyflavor.com/pages/救援小貓sop-1-1 ；https://litomon.com/blog/stray-cat-rescue-guide/]
- **[事實]** 中途之家是無給職志工性質，依賴少數協會補助（咪可思、新屋貓屋、TSPCA 等提供物資/醫療補助）。[來源: https://vocus.cc/article/5ed65fbefd89780001e5b37e ；https://www.lovecat.org/service.html]
- **[軼事]** 高雄等地論壇長年有「請問哪裡有中途之家、有流浪貓需要幫忙」的求助文。[來源: http://www.supervr.net/catbbs/topics/list/99520.page]
- **[推論]** 中途量能無任何全國性媒合系統，全靠 FB 社團/私訊人脈；「奶貓季（春夏）撿到幼貓→中途全滿→民眾不知所措」是年復一年的尖峰痛點。
- **[研究限制]** 「中途之家荒」無官方量化統計（缺口數字不存在），只能以衛教文與論壇求助文間接佐證。

### 3. TNR 志工生態與痛點
- **[事實]** 窩窩深度報導指出 TNR 政策處於「民怨、減量、動物福利」多重壓力下：回置遭居民、里長、民代反對；不回置則收容所無法運作——「抓與不抓、放與不放都會遇到衝突」。[來源: https://wuo-wuo.com/report/issue/2138-tnvrissue]
- **[事實]** 誘捕實務依賴餵養人配合斷食：「明天要抓狗、今天要餓狗，十個有八個做不到」，志工與愛媽協作是成敗關鍵（「成也愛媽，敗也愛媽」）。[來源: https://www.tnr.life/training/teamwork ；https://www.peopo.org/news/822600]
- **[事實]** 餵養爭議：餵食使犬群聚集、與人類活動重疊，產生環境髒亂、人犬衝突、數量增加風險；連資深 TNR 志工亦有人贊成禁餵。[來源: https://wuo-wuo.com/topics/stray-animals/feed-management/1226-feedstraydogs ；https://e-info.org.tw/node/236945]
- **[事實]** 修法草案曾提「餵食照顧志工制度」，欲讓愛媽有合法身分與保險。[來源: https://www.peopo.org/news/822600]

### 4. 私人狗場/愛媽超載
- **[事實]** 趙媽媽狗園收容近 2,000 隻流浪犬貓，負責人近 80 歲，長期面臨飼料短缺與龐大醫療開銷，並有「我走了之後牠們怎麼辦」的接班危機。[來源: https://handhand.org/projects/zhaomamashelter]
- **[事實]** 失控案例：桃園觀音愛媽狗園遭查獲大量犬隻遺骸，132 隻狗多數瘦到肋骨外露、未植晶片未絕育，全數沒入、負責人移送法辦（2022）。[來源: https://www.mirrormedia.mg/story/20220902edi003]
- **[事實]** 太報專題〈「浪浪還活著，我卻死了」〉指出：私人狗園始於善意，但數量膨脹後超出負荷，陷入永續募款困境，捐款「不治本」。[來源: https://www.taisounds.com/w/TaiSounds/society_19100313401409633]
- **[推論]** 公立爆滿 → 壓力外溢到私人狗園/愛媽 → 私人端再超載失控，形成惡性循環；「把撿到的動物轉給愛媽」其實常是把問題往更脆弱的環節推。

### 5. 受傷流浪動物送醫費用：誰出錢
- **[事實]** 原則上「誰送醫誰付費」，醫療費動輒上萬，常超出善心民眾負擔；解法包括動保機關合作醫院（部分縣市撥 1999/1959 可送合作醫院由公費處理）、動物福利基金會急難補助、募款（易生糾紛）、獸醫自行打折。[來源: https://www.threads.com/@im.rron/post/C9uHseVTjud ；https://lkawf.eoffering.org.tw/contents/news_ct?id=15 ；https://www.ptt.cc/bbs/cat/M.1542709526.A.3B4.html ；https://www.tnrtw.org/assist_report.php]
- **[軼事]** Threads 上有獸醫相關爭論：「除非對方全額付錢救，不然根本不該免費或打折救任何流浪貓狗」——顯示獸醫端對長期吸收浪浪醫療成本的反彈。[來源: https://www.threads.com/@thaimelife/post/C_VqwiiBjgj]
- **[推論]** 「各縣市是否有公費合作醫院、補助申請窗口」資訊極度分散，民眾在送醫當下幾乎不可能即時查到——是 bot 高價值的即時回覆內容。

### 6. 認養率與棄養
- **[事實]** 2023 年全國犬貓登記 232,196 隻，高於同年新生兒 135,571 人；全國家犬貓推估 279 萬隻。[來源: https://www.cna.com.tw/news/ahel/202410160068.aspx ；https://www.moa.gov.tw/theme_data.php?theme=news&sub_theme=agri&id=9418]
- **[事實]** 飼主可循「不擬續養」程序付費將寵物交給公立收容所（變相合法棄養管道），北市設「動物讓渡專區」並公告不擬續養動物。[來源: https://www.tcapo.gov.taipei/cp.aspx?n=B967AF09B71C09D8]
- **[軼事]** Dcard 有「不送養了我不養我就丟收容所吧」之類貼文，反映棄養心態與送養挫折。[來源: https://www.dcard.tw/f/pet/p/257087176]
- **[研究限制]** 全國認養率/棄養件數逐年數字未能自搜尋摘要直接取得，原始數據在 https://data.gov.tw/dataset/41236 與動保資訊網，建議後續以 CSV 直接下載核實。

## 三、對 Animal SOS 的意涵

1. **「撿到貓狗 → 接下來怎麼辦」決策樹**是最高頻需求：先判斷是否需介入（奶貓是否真的失親）→ 是否可送公立（各縣市精準捕捉政策）→ 中途/協會清單 → TNVR 選項 → 醫療費補助管道。官方與民間都沒有整合版。[推論]
2. **醫療費資訊即時化**：bot 應內建「縣市 × 公費合作醫院/急難補助」對照，回覆「先打 1999/1959 問合作醫院，自行送醫前先確認誰付費」，可避免民眾事後背債與糾紛。[推論]
3. **奶貓季（約 4–9 月）可預期尖峰**，bot 內容與量能應對齊季節。[推論]
4. **期望管理與轉介誠實度**：不要把民眾無腦導向已超載的愛媽/私人狗園；建議搭配「自己當短期中途」的衛教（保溫、餵食 SOP），降低棄置與二次傷害。[推論]
5. 公開資料集（data.gov.tw 41236、85903）可讓 bot 引用即時收容量能與認養資訊。[事實/推論]

## 五、研究限制

1. 收容所在養量/容留比數字取自報導者等媒體摘要，未核原始開放資料 CSV。
2. 「中途之家缺口」「TNR 志工人數」無官方統計，相關痛點以質性來源佐證。
3. 認養率與棄養逐年統計未取得，僅標示資料所在位置。
